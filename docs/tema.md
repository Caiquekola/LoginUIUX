Trabalho I IHC - 2026
Tema: Engenharia de Usabilidade e Desenvolvimento front-end

Implementar o front-end de um sistema WEB de uma plataforma de venda de ingressos para eventos.
Aplicar todas as Heurísticas de Nielsen da Engenharia de Usabilidade.
Aplicar ferramentas, frameworks e tecnologias de ponta utilizados no mercado (pesquise sobre).


Requisitos mínimos:
Realizar login na plataforma.
Escolher eventos para comprar ingressos.
Escolher entre diferentes categorias de ingressos (Exemplo: normal, área VIP etc).
Escolher lugar (marcar assentos ocupados).
Realizar pagamento e confirmação (simulados)
Listar eventos e ingressos adquiridos no perfil do usuário.
Mostrar em ordem cronológica.
Separar eventos que já passaram.
Disponibilizar um ingresso digital com QR-Code para o usuário.

Observação: 
Não é preciso implementar a persistência em Banco de Dados, apenas o front-end.
Os dados podem ser simulados em vetores simples.
Pode utilizar ferramentas de Inteligência Artificial Generativa em toda a implementação (mas não no relatório final).


O que entregar:
Relatório em PDF contendo 
Identificação dos estudantes.
Apresentação dos objetivos do sistema.
Para cada Heurística: imagens das telas e explicações de como a heurística foi abordada.
Link para um vídeo do sistema em funcionamento (não é obrigatório ter áudio explicativo).
Código fonte do sistema.

Pode ser feito em duplas.
Data da entrega: 04/05.
Valor: 25 pts.
